  • 工具

  1. calc符号中间需要有空格隔开，父元素没有显式高度时，是无法用calc百分比设置子元素的高度。

  2. 单独设置图片宽高是按等比例调整的，如果要更改图片比例可以同时设置宽高。

  3. 当margin和padding的所有方向，当设置的值是百分比时，都是相对最近父级块级元素的width，

  4. 相邻兄弟元素之间相邻外边距会取最大值而不是取和。

  5. 父子垂直外边距相邻了，则子元素的外边距会设置给父元素，都设置了取最大值。

    可以给父元素添加一个边框 可以给父元素添加一个内边距 可以设置父元素 overflow：hidden； clear 定位

  6. box-shadow:水平距离 垂直距离 阴影距离 阴影大小 阴影颜色 内外阴影(默认outset，不用写) box-shadow:20px 20px 20px 20px rgba(0,0,0,0.5) inset 支持逗号分隔语法，可以创建任意数量的投影。层层叠加. box-shadow: 0 0 0 10px #655,0 0 0 15px deeppink; outline outline也可以产生边框，但是不能叠加，但是可以通过outline-offset设置负值实现在内部添加边框 outline: 5px solid deeppink; outline-offset:-10px

  7. 解决输入法会把底部导航栏顶上来的问题：监听窗口高度变化就设置body高度

var oHeight = document.body.clientHeight; //保存浏览器默认高度
window.onresize = function() {
	if(document.body.clientHeight<oHeight){
		document.getElementsByTagName('body')[0].style.height =oHeight+'px'; 
	}
}
  1. 1em = 1font-size 当前元素字体大小，不过需要注意 font-size会继承的
  2. word-break：break-all/keep-all(默认)；句子长度超出块元素区域，单词是否拆开两行显示，有连字符(-)也会拆开 white-space:normal/nowrap;nowrap使文字超出块元素区域时强制在一行显示而不是换行，除非遇到br text-overflow：ellipsis；如果文字超出块元素，最后会显示一个省略号(开启了overflow：hidden和white-space：nowrap)
  3. visibility:hidden 隐藏不显示，但是是会依然占据页面位置(而diplay会不再占据)
  4. 外部样式 通过link标签加载外部样式表，必须放在head元素中。可以通过title属性增加候选样式表。 @import声明必须放在所有样式表开头，此外无限制。 @impot url(sheet2.css) screen; 如果一个外部样式表要用到另一个外部样式表，可以用@import指令。因为没法用link。
  5. 都开启绝对定位，后面的兄弟元素会覆盖前面的元素
